Checkout

Vendr Checkout provides a ready made checkout flow for Vendr,
the eCommerce solution for Umbraco v8+

Getting Started

To get started with Vendr Checkout we first need to install all the related content.
By clicking the Install button below Vendr Checkout will install all the Data Types,
Doc Types and Content nodes needed.

If you have installed Vendr Checkout before, the installer will also perform the relevant upgrades.

NB: Nothing will be removed as part of an upgrade.
